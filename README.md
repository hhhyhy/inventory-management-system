# Inventory Management System

Spring Boot와 MySQL을 활용한 재고 관리 시스템입니다.

## 프로젝트 개요
상품 정보를 등록하고 재고 수량을 관리할 수 있는 웹 기반 관리 시스템입니다.

## 주요 기능
- 상품 등록
- 상품 수정 및 삭제
- 재고 수량 관리
- 입고 및 출고 내역 관리
- 상품 검색

## 사용 기술
- Java
- Spring Boot
- MySQL
- JPA
- Thymeleaf

## 학습 내용
- Spring Boot MVC 구조
- 데이터베이스 연동
- CRUD 기능 구현
- 관리자 페이지 구성
